Description of the EE-Note:

	This application note discusses about the features, performance and usage of ADSP-2156x FIR/IIR Accelerators.


*********************************************************************************************************************************
Code Details: 

ADSP_2156x_FIRA_Performance	:	This code can be used to measure ADSP-2156x FIRA performance for different filter parameters. 
ADSP_SC57x_FIRA_Performance	:	This code can be used to measure ADSP-SC57x FIRA performance for different filter parameters. 
ADSP_214xx_FIRA_Performance	:	This code can be used to measure ADSP-214xx FIRA performance for different filter parameters. 
ADSP_2156x_FIR_Core_Performance	:	This code can be used to measure ADSP-2156x Core FIR performance for different filter parameters. 
ADSP_2156x_IIRA_Performance	:	This code can be used to measure ADSP-2156x IIRA performance for different filter parameters. 
ADSP_SC57x_IIRA_Performance	:	This code can be used to measure ADSP-SC57x IIRA performance for different filter parameters. 
ADSP_214xx_IIRA_Performance	:	This code can be used to measure ADSP-214xx FIRA performance for different filter parameters. 
ADSP_2156x_IIR_Core_Performance	:	This code can be used to measure ADSP-2156x Core FIR performance for different filter parameters. 
FIR_Multi_Channel_Processing	:	This code can be used as an example to understand the usage of ADSP-2156x FIRA Device Driver. 
IIR_Multi_Channel_Processing	:	This code can be used as an example to understand the usage of ADSP-2156x IIRA Device Driver.  
ADSP_2156x_FIRA_Driver_Benchmark:	This code can be used to measure ADSP-2156x FIRA Device Driver Benchmarks. The Driver benchmarks were measured with CCES 2.9.0 w1904151.
ADSP_2156x_IIRA_Driver_Benchmark:	This code can be used to measure ADSP-2156x IIRA Device Driver Benchmarks. The Driver benchmarks were measured with CCES 2.9.0 w1904151.
Direct_Replacement		: 	This code can be used as an example to undsertand the real time implementation of "Direct Replacement" Accelerator Usage Model.
Pipelined			:	This code can be used as an example to undsertand the real time implementation of "Data Pipelined" Accelerator Usage Model.
Split_Task			:	This code can be used as an example to undsertand the real time implementation of "Split Task" Accelerator Usage Model.
